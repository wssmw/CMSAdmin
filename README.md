## 项目简介

Admin 是一个基于 Vue3 + Vite + TypeScript 的前端类似 discord 交友聊天项目，使用了 Vue Router、Vuex、Element Plus 等主流框架和库，项目采用了模块化、组件化、工程化的开发模式。

## 快速上手

- **Clone：**

```text
# GitHub
git clone https://github.com/wssmw/CMSAdmin.git
```

- **Install:**

```text
npm install 
```

- **Run:**

```text
npm run start 
```

- **Build:**

```text
npm run build 
```
